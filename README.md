##  Описание приложения

**Scan** - это мобильное приложение, которое позволяет:
- **Сканировать QR-коды и штрих-коды** с помощью камеры устройства
- **Искать данные** в загруженных Excel/CSV файлах по отсканированным кодам или текстовому запросу
- **Просматривать результаты** в удобном формате
- **Редактировать найденные данные** и сохранять изменения в локальные файлы
- **Управлять файлами данных** - загружать, заменять, очищать

##  Основные функции

###  Поиск данных
- **Сканирование кодов**: автоматический поиск данных по отсканированным QR/штрих-кодам
- **Текстовый поиск**: ручной ввод поисковых запросов (минимум 2 символа)
- **Умный поиск**: поиск по всем полям загруженных данных
- **Множественные результаты**: выбор из списка при нескольких совпадениях

###  Работа с файлами
- **Поддержка форматов**: Excel (.xlsx, .xls) и CSV файлы
- **Автоопределение**: автоматическое определение типа файла
- **Кодировка**: поддержка Cp1251 для CSV файлов
- **Локальное хранение**: кэширование данных в локальной базе данных

###  Просмотр и редактирование
- **Форматированный вывод**: данные отображаются в читаемом формате
- **Редактирование**: возможность изменять найденные данные
- **Сохранение**: автоматическое сохранение изменений в папку Documents
- **История**: каждый сеанс редактирования помечается временной меткой

##  Архитектура

Приложение построено на современной архитектуре **MVVM** (Model-View-ViewModel) с использованием:

###  Основные компоненты

**Data Layer (Слой данных):**
- `DataRepository` - центральный репозиторий для управления данными
- `AppDatabase` - локальная база данных Room для кэширования
- `ExcelDataDao` - Data Access Object для работы с базой данных
- `DataFileReader` / `ExcelDataFileReader` - утилиты для чтения файлов

**Domain Layer (Бизнес-логика):**
- `MainViewModel` - главная ViewModel для управления состоянием приложения
- `ScannerViewModel` - логика работы сканера
- `SearchViewModel` - логика текстового поиска
- `DataRow` - модель данных для представления строк

**Presentation Layer (UI):**
- `MainActivity` - главная активность с навигацией
- `ScannerFragment` - экран сканирования кодов
- `SearchFragment` - экран текстового поиска
- `ResultsFragment` - экран отображения результатов
- `EditFragment` - экран редактирования данных

##  Технологии

- **Языки**: Kotlin , Java
- **Минимальная версия Android**: API 21 (Android 5.0)
- **Архитектура**: MVVM с LiveData/StateFlow
- **Локальная база данных**: Room Persistence Library
- **Сканирование кодов**: ZXing через библиотеку JourneyApps
- **Работа с Excel**: Apache POI
- **Асинхронность**: Kotlin Coroutines
- **Навигация**: Fragment + BottomNavigationView
- **Сборка**: Gradle с Kotlin DSL
